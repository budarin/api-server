API server with OAuth authorization

Используемые технологии и библиотеки

Сервер:
- Koa.js - сервер
- Koa-router - роутер
- koa-oauth-server - oauth 2.0 server
- winston - журналирование

Билиотеки и инструменты разработки
- babel - ES7 транспилятор
